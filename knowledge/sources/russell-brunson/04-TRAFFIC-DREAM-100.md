## Filosofia Central

Your job is not to create traffic -- traffic already exists. People are already on Facebook, blogs, email lists, YouTube. Your job is to figure out where they are and divert them to you. Once you understand this, the entire game changes. You stop trying to manufacture attention from thin air and start redirecting rivers that already flow.

The biggest goal you can focus on is building a distribution channel. If you have distribution, you can do anything -- sell any product, sell ads, do sponsorships. Distribution is the ultimate asset.

And here is the thing most people miss: all traffic should flow to one destination -- your email list. You want to convert every type of traffic into traffic you OWN. Because traffic you own is the only traffic that is truly yours.

### Traffic Temperature

Different traffic temperatures require different approaches. You cannot send cold traffic to the same page as hot traffic.

```
+-----------------------------------------------------------------+
|                TRAFFIC TEMPERATURE                                |
+-----------------------------------------------------------------+
|                                                                   |
|   HOT TRAFFIC                                                    |
|   +----------------------------------------------------------+  |
|   |  Product aware. They know you. They know your offer.       |  |
|   |  -> Send directly to sales page or webinar                |  |
|   |  -> Shortest funnel needed                                |  |
|   |  -> Dream 100 referrals, your email list                  |  |
|   +----------------------------------------------------------+  |
|                                                                   |
|   WARM TRAFFIC                                                   |
|   +----------------------------------------------------------+  |
|   |  Solution aware. They know the category but not you.       |  |
|   |  -> Need pre-frame bridge (quiz, article, video)          |  |
|   |  -> Medium funnel depth                                   |  |
|   |  -> Retargeting, lookalike audiences                      |  |
|   +----------------------------------------------------------+  |
|                                                                   |
|   COLD TRAFFIC                                                   |
|   +----------------------------------------------------------+  |
|   |  Problem aware or completely unaware. Don't know you.      |  |
|   |  -> Need longest funnel (education first)                 |  |
|   |  -> Use "web class" not "webinar"                         |  |
|   |  -> Headlines: 3-4 words maximum                          |  |
|   |  -> Images with eyes increase CTR                         |  |
|   |  -> Inverted/negative images create pattern interrupt      |  |
|   |  -> Average person needs 7 touches before purchase        |  |
|   +----------------------------------------------------------+  |
|                                                                   |
|   CRITICAL: The pre-frame through which someone arrives          |
|   at your page completely changes their behavior.                |
|   All clicks are NOT created equal.                              |
|                                                                   |
+-----------------------------------------------------------------+
```

### Internet Fame vs Real Fame

You want to become internet famous, not real famous. Internet famous means a specific segment of people know you, like you, love you, and will pay premium prices for your expertise. A homeless man with 300,000 YouTube followers on juicing charged $500/hour for consulting. Platform plus attractive character equals premium pricing.

## Armadilhas

### Trying to Create Traffic From Scratch

Traffic already exists. People are already spending hours on Facebook, YouTube, podcasts, blogs. Your job is to redirect it, not create it. Stop trying to build an audience from zero when there are Dream 100 partners with audiences ready to hear your message.

### Spreading Across All Channels

Pick ONE distribution channel plus email and go deep. Only after approximately 1 million followers add a second. The person who tries to be everywhere is nowhere. Go an inch wide and a mile deep.

### Undervaluing Email

At minimum email your list once a day. If you do not email for two weeks, response rate drops by 50%. I went from monthly to daily emails, and each increase more than doubled income. Under-emailing is the default mistake. Email is the only channel you truly OWN.

### Ignoring the Pre-Frame

The context through which someone arrives at your content shapes their entire experience. One word change on a squeeze page changed conversion and downstream behavior dramatically. All clicks are not created equal. The pre-frame through which someone arrives at your page completely changes their behavior.
