## METODOLOGIAS

### MET-30DC-040: Lead Graph Matrix (Day 27)

**Como usar para decisão de estratégia de lead gen:**

**PASSO 1 — Mapear Tipo de Lead:**
- Definir qual oferta de entrada você tem
- Mapear onde cai na matriz (Desejo × Consumo)

**PASSO 2 — Calcular CPL por Quadrante:**
- PDF Download: mais barato por lead
- Webinar: médio, mas lead mais qualificado
- Sales Call: mais caro, menos qualificado

**PASSO 3 — Otimizar Funil:**
- Usar PDF como entrada → upsell para webinar
- Webinar como qualificador → sales call apenas para quem assistiu
- Nunca levar lead frio direto para sales call

**PASSO 4 — Canais por Quadrante:**
- PDF/Ebook: Facebook, Instagram
- Webinar: YouTube, Email list, LinkedIn
- Sales Calls: Outbound direto (30DC methodology)

> ^[30DC-0027:L34-167]

### MET-30DC-041: Loom Screen Capture Sales System (Day 27)

**Uso de Loom para vídeos de prospecção e follow-up:**

**Setup técnico:**
- Screen recording + webcam no canto
- Prospect vê você + vê a tela simultaneamente
- Cria sensação de call personalizada sem precisar agendar

**Tipos de vídeo Loom:**
1. **Audit video:** Gravar análise da presença digital do prospect
2. **Personalizado:** Mostrar dados específicos do negócio deles
3. **Follow-up:** Atualizar sobre algo novo que encontrou

**Regra de retenção:**
- 70-90% de retenção quando você ENSINA algo via Loom
- O prospect aprende → você retém o conhecimento também
- Cria autoridade instantânea

> ^[30DC-0027:L167-212]

### MET-30DC-042: Outsourcing 101 — Duplicação de Operações (Day 28)

**Framework de delegação para múltiplas empresas/clientes:**

**FASE 1 — Documentar Processos:**
- Gravar Loom de cada tarefa repetitiva que você faz
- Criar SOP (Standard Operating Procedure) em texto
- Biblioteca de SOPs = base para contratar

**FASE 2 — Identificar o Que Delegar:**
```
DELEGAR IMEDIATAMENTE:
├── Pesquisa de prospects (VAs)
├── Envio de emails (assistente)
├── CRM management
├── Relatórios de clientes
└── Admin em geral

MANTER VOCÊ:
├── Estratégia
├── Calls de fechamento
├── Relacionamento com clientes
└── Desenvolvimento de produto/serviço
```

**FASE 3 — Estrutura de Time:**
```
FOUNDER (você)
    ↓
PROJECT MANAGER (operações diárias)
    ↓
SALES PEOPLE (prospecção + follow-up)
    ↓
DELIVERY (execução dos serviços)
```

**FASE 4 — Plataformas para Freelancers:**
- Upwork, Freelancer.com
- VA agencies (especializadas)
- LinkedIn para sales reps

### MET-30DC-043: Pipeline Visual Management (Day 29)

**Como organizar e gerenciar pipeline visualmente:**

**Estrutura da linha:**
```
HOTTIES → SUB-HOTTIES → LUKEWARMS → COLD
[Prontos]  [Aquecendo]  [Mornos]    [Frios]

MOVIMENTO: Prospect pode mover em qualquer direção.
           Follow-up criativo move COLD → LUKEWARM.
           Objection handling move LUKEWARM → SUB-HOTTIE.
           Urgency/close move SUB-HOTTIE → HOTTIE.
```

**Multi-channel por stage:**
```
HOTTIES:     Ligação, WhatsApp, Zoom imediato
SUB-HOTTIES: Email + Vídeo + Texto
LUKEWARMS:   Email criativo + Props físicos
COLD:        Props extremos + Última tentativa formal
```

**Regra de pipeline health:**
- Revisar pipeline DIARIAMENTE (10 min)
- Mover prospects baseado na última interação
- Nunca deixar prospect parado no mesmo stage por > 7 dias

> ^[30DC-0029:L34-134]

### MET-30DC-044: Lead Sources & Verification (Day 30)

**Ferramentas para encontrar e verificar leads:**

**Para encontrar:**
- **Hunter.io:** Encontrar emails corporativos por domínio
- **LinkedIn:** Prospecção por cargo + empresa + localização
- **Facebook Messenger:** Prospecção via grupos e páginas
- **WHOIS:** Encontrar dono de domínio/website

**Para verificar:**
- **Email verification tools:** Verificar se email existe antes de enviar
- **Hunter.io verification:** Built-in verificação de deliverability
- **Neverbounce:** Verificação em massa
- **ZeroBounce:** Alternativa premium

**Regra:**
- Nunca enviar para lista não verificada (destrói sender reputation)
- Verificar ANTES de importar no CRM
- Limpeza mensal da lista existente
