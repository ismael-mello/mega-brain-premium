## Thumbnail Psychology — Framework 3C

**Fonte:** CA-0016 ^[knowledge/sources/client-accelerator/03-THUMBNAIL-PSYCHOLOGY.md]

```
+------------------------------------------------------------------------------+
| FRAMEWORK 3C — CRITÉRIOS DE UM THUMBNAIL GANHAR CLIQUES                     |
+------------------------------------------------------------------------------+
|  C1 — CLARITY    Entender o tópico em < 0.5 segundo                        |
|                   Legível em mobile (70% das views são mobile)              |
|                   Teste: Flash Test (1 segundo de exposição)                |
+------------------------------------------------------------------------------+
|  C2 — CONTRAST   Contraste visual: cores, luz e sombra                     |
|                   Contraste emocional: before/after, problema vs solução    |
|                   Contraste conceitual: expectativa vs realidade            |
+------------------------------------------------------------------------------+
|  C3 — CURIOSITY  Criar tensão psicológica sem resolver no thumbnail        |
|                   Ambiguidade estratégica: lacuna que só o vídeo preenche   |
|                   NÃO é clickbait — é promessa legítima de valor            |
+------------------------------------------------------------------------------+
```

### 6 Attention Triggers:
1. **Rostos Expressivos** — wiring evolutivo: cérebro prioriza faces
2. **Pessoas Famosas** — reconhecimento instantâneo, autoridade
3. **Números Grandes** — valor quantificado chama mais atenção
4. **Visuais Familiares** — logos, ícones conhecidos
5. **Estética/Beleza** — dopamina visual
6. **Movimento/Perigo/Drama** — gatilhos primitivos de sobrevivência

### 5 Curiosity Gap Archetypes:

| Arquétipo | Mecanismo | Exemplo |
|-----------|-----------|---------|
| Before-and-After | Transformação implícita | Foto antes sem depois |
| Challenge | Suspense/stakes | "Tentei X por 30 dias" |
| Contradiction | Dissonância cognitiva | "Por que parei de [coisa óbvia]" |
| Novelty | O que é isso? | Visual estranho/inesperado |
| Result | Prova aspiracional | Números sem contexto |

### Insights Avançados:
- **Neighbor Effect:** pesquisar top 10 thumbnails do mesmo tópico e fazer diferente
- **Authenticity Advantage:** Real > Polished em muitos nichos
- **Instant Click Phenomenon:** após 5-10 vídeos com padrão consistente, reconhecimento automático
- **Expectation Contract:** Thumbnail cria expectativa — vídeo deve satisfazê-la (violação = penalidade de watch time)

### Platform Psychology:
- **YouTube:** Curiosity-First — alta competição, objetivo é CTR
- **Facebook/Instagram:** Hook-Alignment — primeiros 3s do vídeo DEVEM ser consistentes com thumbnail

### Checklist de Qualidade:
- [ ] Clareza em < 0.5 segundos?
- [ ] Contraste visual suficiente?
- [ ] Lacuna de curiosidade criada?
- [ ] Legível em tamanho mobile?
- [ ] Alinhado ao hook dos primeiros 3s do vídeo?
- [ ] Diferente dos thumbnails concorrentes?
- [ ] Consistente com identidade visual?
- [ ] Flash test passou?
