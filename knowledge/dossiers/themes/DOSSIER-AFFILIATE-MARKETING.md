## Filosofia Central

Affiliate marketing is the most leveraged traffic strategy: someone else does the SEO, PPC, and email marketing for you -- you only pay when they make a sale. The key is treating affiliates like partners, not vendors. Pay generously, train them, and make promotion easy.

## Modus Operandi

### Affiliate Army 5-Step Framework ^[FW-RB-065]

```
+-----------------------------------------------------------------------------+
|                AFFILIATE ARMY - 5 STEPS                                     |
+-----------------------------------------------------------------------------+
|                                                                              |
|  1. RECRUIT from Dream 100                                                  |
|     - Target influencers already serving your dream customers              |
|     - Build relationship BEFORE asking (60+ days)                          |
|                                                                              |
|  2. MAKE them official affiliates                                           |
|     - Set up tracking, affiliate portal, swipe copy                        |
|     - Remove all friction from the signup process                          |
|                                                                              |
|  3. GIVE them a reason to promote                                           |
|     - Launches, contests, exclusive bonuses                                |
|     - "What's in it for ME?" must be answered clearly                      |
|                                                                              |
|  4. TRAIN them to become super affiliates                                   |
|     - Provide swipe copy, banners, email templates                         |
|     - Teach them the highest-converting promotional strategies             |
|                                                                              |
|  5. COMPENSATE generously                                                   |
|     - Pay ABOVE market rates                                               |
|     - Sticky cookies, high commissions, cash prizes                        |
|     - "They don't need you; you need them" ^[PHI-RB-072]                   |
|                                                                              |
|  SOURCE: Traffic Secrets p.313-320                                         |
+-----------------------------------------------------------------------------+
```

### Three Lost Funnels (Network Marketing) ^[FW-RB-067]

Three proven offline selling methods adapted for online:

1. **Bridge Funnel** - Mimics three-way calls where upline bridges new distributor to prospect
2. **Home Party Funnel** - Mimics live in-home product demos with FOMO and social proof
3. **Hotel Meeting Funnel** - Mimics weekly presentation events that recruit leaders (not just product lovers)

SOURCE: Network Marketing Secrets p.22-28

### Billy Mays Demo Script ^[FW-RB-068]

6-step product demonstration framework:
1. Introduce the product
2. **Faster and Easier** -- show how YOUR product beats alternatives
3. **Eliminate Alternatives** -- unique selling point
4. **Introduce Price** with anchor comparison
5. **Urgency and Scarcity**
6. **Risk Reversal** -- guarantee

SOURCE: Network Marketing Secrets p.44-46

## Armadilhas

### Taking Rejection Personally
Affiliate recruitment is a numbers game. Don't take rejection personally. Double your ask list to hit target. ^[HEUR-RB-104]

### Low Commission Rates
Pay above market. Affiliates have unlimited options. Your commission must be compelling enough to earn their promotional energy. ^[PHI-RB-072]

### No Training/Support
Don't just give affiliates a link. Give them swipe copy, banners, email templates, and strategies. Train them. ^[FW-RB-065]
