## Filosofia Central

Traffic is not about "getting" visitors -- it's about understanding WHERE your dream customers already congregate and either working your way in (free) or buying your way in (paid). The Dream 100 strategy is the foundation: a small number of players control the majority of attention in any market. Every platform follows a lifecycle (Adoption→Price Hike→Slap), so you must build your own distribution (email list) that you own.

## Modus Operandi

### Fill Your Funnel Framework (6 Steps) ^[FW-RB-053]

```
+-----------------------------------------------------------------------------+
|               FILL YOUR FUNNEL - 6 REPEATABLE STEPS                        |
+-----------------------------------------------------------------------------+
|                                                                              |
|  Apply to EVERY platform (Instagram, Facebook, YouTube, Google, Podcast):   |
|                                                                              |
|  1. UNDERSTAND history and goal of the platform                            |
|  2. FIND your Dream 100 on the platform                                    |
|  3. IDENTIFY publishing strategy and create plan                           |
|  4. WORK your way in (free - earn organic reach)                           |
|  5. BUY your way in (paid - ads targeting Dream 100 audiences)             |
|  6. FILL your funnel (convert traffic to leads/buyers)                     |
|                                                                              |
|  This framework is PLATFORM-AGNOSTIC and FUTURE-PROOF.                     |
|  SOURCE: Traffic Secrets p.143-146, 272-273                                |
+-----------------------------------------------------------------------------+
```

### Hook, Story, Offer ^[FW-RB-052]

```
+-----------------------------------------------------------------------------+
|                      HOOK, STORY, OFFER                                     |
+-----------------------------------------------------------------------------+
|                                                                              |
|  EVERY piece of marketing must have:                                       |
|                                                                              |
```

### Searcher vs Scroller ^[MM-RB-057]

```
+-----------------------------------------------------------------------------+
|                    SEARCHER vs SCROLLER                                      |
+-----------------------------------------------------------------------------+
|                                                                              |
|  Two types of online behavior requiring DIFFERENT strategies:              |
|                                                                              |
|  SEARCHERS                          SCROLLERS                               |
|  - Actively looking for solutions   - Passively browsing feeds             |
|  - Google, YouTube search           - Facebook, Instagram, TikTok          |
|  - Intent-based targeting           - Interest-based targeting             |
|  - SEO, PPC, keyword ads            - Hook-driven interruption ads         |
|  - Higher intent, smaller volume    - Lower intent, massive volume         |
|                                                                              |
|  Different hooks, different funnels for each behavior type.                |
|  SOURCE: Traffic Secrets p.42-44                                           |
+-----------------------------------------------------------------------------+
```

### 3 Closes in Follow-Up Funnels ^[FW-RB-054]

```
+-----------------------------------------------------------------------------+
|              3 CLOSES IN FOLLOW-UP FUNNELS                                  |
+-----------------------------------------------------------------------------+
|                                                                              |
|  Close #1: EMOTION (50% of buyers)                                         |
|  - Storytelling, identity, emotional connection                            |
|  - Lead with this in your follow-up sequence                               |
|                                                                              |
|  Close #2: LOGIC (30% of buyers)                                           |
|  - Case studies, data, ROI calculations                                    |
|  - Follow emotion with rational justification                              |
|                                                                              |
|  Close #3: FEAR (20% of buyers)                                            |
|  - Urgency, scarcity, deadlines                                           |
|  - Final push for procrastinators                                          |
|                                                                              |
|  Structure follow-up sequences in this exact order.                        |
|  SOURCE: Traffic Secrets p.118-120                                         |
+-----------------------------------------------------------------------------+
```

### 3 Questions for Any New Platform ^[FW-RB-066]

1. What is the GOAL of the platform?
2. What strategy can I use to HELP the platform with its goal?
3. What tactics is the platform REWARDING people for right now?

Align your strategy with the platform's goals = rewarded with reach. ^[Traffic Secrets p.204-205]

### Publishing Plans

**Instagram Daily:** 2 JK5 images (hooks), 10-30 Stories, 1 swipe-up CTA Story, engage Dream 100 (10 comments + 10 DMs). Weekly: 1 IGTV. Monthly: 1 Highlights Mini Webinar. ^[MET-RB-032]

**Facebook Daily:** 1 personal profile post (JK5, Hook→Story→Offer, link in comments). Weekly: 2-3 Fan Page value videos + 1 Live Perfect Webinar + 1 Group hangout. Always: $10-$20 boost on every Fan Page post. ^[MET-RB-033]

### Dream 100 Execution Process ^[MET-RB-029]

1. Identify top influencers/brands in your submarket
2. Follow them on each platform
3. Study their content daily (3-5 min)
4. Like/comment on 10+ posts daily
5. DM at least 10 of their videos daily
6. Build genuine relationship for 60+ days before asking for anything

### Retargeting Setup ^[MET-RB-031]

1. Install pixel on all funnel pages
2. Create Engaged audience (5-day window)
3. Create Landed audience (7-day window)
4. Create Owned audience (leads + buyers)
5. Run Emotion→Logic→Fear ads to each bucket

## Armadilhas

### Building on Rented Land
