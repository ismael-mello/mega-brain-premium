## Filosofia Central

Funnels are the operating system of online business. Every "new" funnel is just three core funnels bolted together in different combinations, like Legos. The front-end exists to break even and build a list -- ALL real profit comes from middle and backend funnels. The company that can spend the most to acquire a customer wins.

### Funnel Diagnostic Framework ^[FW-RB-023]

```
+-----------------------------------------------------------------------------+
|                     FUNNEL DIAGNOSTIC FRAMEWORK                              |
+-----------------------------------------------------------------------------+
|                                                                              |
|  NEVER scrap a whole funnel. Diagnose metric by metric.                     |
|                                                                              |
|  Metric 1: Squeeze page % -> low? Fix the pre-frame/hook                   |
|  Metric 2: Front-end buy % -> low? Fix the offer/price                     |
|  Metric 3: OTO1 % -> low? Fix the upsell (next step, not same)             |
|  Metric 4: OTO2 % -> low? Fix the downsell                                 |
|  Metric 5: Email open % -> low? Fix subject lines                          |
|                                                                              |
|  Find the ONE broken piece. Fix ONLY that.                                  |
|  Usually 2-3 iterations to working funnel. ^[HEUR-RB-034]                   |
|                                                                              |
+-----------------------------------------------------------------------------+
```

## Armadilhas

### Trying to Profit on Front-End
The front-end is for BREAKING EVEN and building a list. All profit comes from middle and backend. ^[PHI-RB-005]

### Selling More of the Same as Upsell
When someone buys on a topic, the itch is scratched. Upsell the NEXT STEP, never more of the same. ^[MM-RB-039]

### Closing the Loop
"Your order is complete" kills upsell conversions. Keep the loop open: "Wait, your order is not yet complete." ^[MM-RB-040]

### Scrapping Funnels
Never scrap a whole funnel. Find the ONE broken metric and fix it. 2-3 iterations. ^[HEUR-RB-034]

### Building 30 Funnels Before One Works
Pick ONE funnel type and master it until $1M. Then diversify. ^[HEUR-RB-006]

### Ignoring Follow-Up
$1 front-end generates $16 in follow-up. The communication funnels are the glue. ^[HEUR-RB-001]

### OTO Upsell Script (11 Steps) ^[MET-RB-020]

1. Congrats on ordering
2. 3X 2X (triple results, double speed)
3. Smart/Why (you made a smart choice because...)
4. Question 3X 2X
5. Exclusive (NOT for everyone)
6. Results/Fast/Speed
7. The 1 Thing (cool name for upsell)
8. Future Casting
9. CTA
10. Guarantee
11. Value Stacking + Scarcity -> 2nd CTA

SOURCE: Module 8 Sales Scripts Workbook ^[DotCom Secrets p.116-117]

### Break-Even Funnel Optimization ^[MET-RB-030]

```
+-----------------------------------------------------------------------------+
|              BREAK-EVEN FUNNEL OPTIMIZATION (ACV Calculation)               |
+-----------------------------------------------------------------------------+
|                                                                              |
|  Product price + Order bump + OTO1 + OTO2 = Average Cart Value (ACV)       |
|                                                                              |
|  EXAMPLE (DotCom Secrets book funnel):                                     |
|  Book $7.95 + Audiobook $37 (16%) + Masterclass $97 (5%)                   |
|  + Funnel Scripts $297 (2%) = ACV $37.71 vs CPA $23                        |
|  = $14.71 PROFIT per customer                                              |
|                                                                              |
|  IF ACV > CPA → You win. Scale infinitely.                                 |
|  SOURCE: Traffic Secrets p.87-89                                           |
+-----------------------------------------------------------------------------+
```

### The Shadow Funnel ^[MM-RB-060]

The invisible journey people take researching you AFTER seeing your ad but BEFORE buying. They search your name, company, and product in Google. This "shadow funnel" creates traffic you MUST capture with a Funnel Hub. ^[Traffic Secrets p.297-299]

### The Funnel Hub ^[MM-RB-065]

A brand-centric website that organizes all your funnels, captures shadow traffic, and builds credibility. Bridges branding and direct response. Looks like a traditional website for credibility but strategically funnels visitors into your value ladder. ^[Traffic Secrets p.295-301]

### Three Lost Funnels (Network Marketing) ^[FW-RB-067]

Three proven offline selling methods adapted to online:
1. **Bridge Funnel** - mimics three-way calls
2. **Home Party Funnel** - mimics live in-home demos with FOMO
3. **Hotel Meeting Funnel** - mimics weekly presentation events

SOURCE: Network Marketing Secrets p.22-28
