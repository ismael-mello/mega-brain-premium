## CONTRIBUIÇÃO ÚNICA

Michael Hauge traz a perspectiva de **Hollywood profissional** para storytelling em marketing. Enquanto outros experts falam de storytelling como técnica de vendas, Hauge ensina a **estrutura narrativa comprovada** que sustenta filmes de bilhões de dólares — e mostra como a mesma estrutura se aplica à jornada do cliente.

Sua contribuição mais valiosa é o framework **Identity vs Essence** — a compreensão de que toda transformação real requer abandonar o "falso eu" (identidade/armadura emocional) e viver no "eu verdadeiro" (essência). Isso explica por que clientes desistem mesmo quando têm todas as ferramentas.

## FRAMEWORKS PRINCIPAIS

### 1. The 6-Stage Story Structure
```
┌──────────┐  ┌──────────┐  ┌──────────┐  ┌──────────┐  ┌──────────┐  ┌──────────┐
│  SETUP   │→│   NEW    │→│ PROGRESS │→│ COMPLI-  │→│  FINAL   │→│AFTERMATH │
│  0-10%   │  │SITUATION │  │ 25-50%   │  │ CATIONS  │  │  PUSH    │  │ 90-100%  │
│          │  │ 10-25%   │  │          │  │ 50-75%   │  │ 75-90%   │  │          │
└──────────┘  └──────────┘  └──────────┘  └──────────┘  └──────────┘  └──────────┘
      ↑             ↑             ↑             ↑             ↑
    TP1:10%       TP2:25%       TP3:50%       TP4:75%       TP5:90%
  Opportunity   Change of     Point of No    Major         Climax
                  Plans        Return        Setback
```
