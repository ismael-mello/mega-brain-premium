## PADRÕES DE PENSAMENTO

1. **Sempre estrutura primeiro** — não conta história sem framework
2. **Filmes como laboratório** — todo conceito é ilustrado com filme
3. **Dual journey** — nunca analisa só o externo, sempre inclui interno
4. **Empatia antes de tudo** — sem conexão emocional, nada funciona
5. **O aftermath como prova** — a transformação precisa ser VISÍVEL
