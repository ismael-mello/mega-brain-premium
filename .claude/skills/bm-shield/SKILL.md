## Os 13 Passos do BM SHIELD

### ETAPA 1: Criar o Gerenciador de Negócios

1. Acessar **business.facebook.com** → Criar Conta
2. Nome da empresa: **nome real** (conforme CNPJ)
3. Nome do administrador: **nome real**
4. E-mail comercial: **e-mail do domínio** (não Gmail)
5. Preencher dados conforme cartão CNPJ:
   - País: Brasil
   - Endereço: **exatamente** como no CNPJ
   - CNPJ: preencher
   - Telefone: **celular pessoal** (importante para verificação)
   - Site: URL do domínio
6. Tipo: "promove próprios bens e serviços"

### ETAPA 2: Adicionar Administrador Secundário (IMEDIATO)

**PRIMEIRA COISA após criar o BM.**

- Configurações → Pessoas → Adicionar novo administrador
- Pelo menos **2-3 administradores** no total
- **Motivo:** se um perfil for bloqueado, você não perde acesso ao BM

### ETAPA 3: Ativar 2FA (Autenticação de Dois Fatores)

- Central de Segurança → Ativar 2FA para **todos** os administradores
- Cada perfil pessoal: Configurações → Segurança e Login → 2FA

### ETAPA 4: Criar e Configurar Página do Facebook

- Criar a página **DENTRO** do gerenciador de negócios
- Categoria: a que melhor representa (educação, consultoria, etc.)
- Preencher **TUDO**: foto de perfil, capa, história, descrição, endereço, e-mail, telefone
- Configurar lista de palavras negativas para moderação (CSV)

### ETAPA 5: Criar Conta de Anúncio

- Criar **DENTRO** do gerenciador (não usar conta pessoal)
- **ATENÇÃO**: moeda NÃO pode ser trocada depois → selecionar **Real (BRL)**
- Fuso horário: Brasília
- Atribuir administradores

### ETAPA 6: Criar Aplicativo no Facebook Developers

- Ir em **developers.facebook.com** → Meus Aplicativos → Adicionar
- Nome de exibição: nome do projeto
- E-mail de contato: e-mail comercial
- Adicionar o aplicativo ao gerenciador de negócios
- **Motivo:** a função de verificação do BM só é ativada após TODOS os ativos configurados, incluindo aplicativo

### ETAPA 7: Conectar Conta do Instagram

- Configurações → Contas → Instagram
- Conectar conta **profissional** do Instagram
- Interligar com conta de anúncio (ativos conectados)

### ETAPA 8: Criar Pixel

- Criar o pixel **PELO gerenciador** (não pela conta de anúncio)
- Vantagem: pode compartilhar para qualquer conta ou parceiro
- Um pixel pode ser usado em múltiplos sites/produtos
- Atribuir pixel à conta de anúncio e linha de negócios

### ETAPA 9: Verificar Domínio

- Configurações → Segurança da Marca → Domínios → Adicionar
- Método preferido: **DNS** (mais simples que arquivo HTML)
- Adicionar registro **TXT** no DNS do domínio (Cloudflare, HostGator, etc.)
- Apontar para @ (raiz)
- Valor: código fornecido pelo Facebook
- Cloudflare propaga quase instantaneamente
- **Atribuir página ao domínio verificado**

### ETAPA 10: Configurar Forma de Pagamento no BM

- Adicionar cartão/PayPal/boleto **NO gerenciador** (não na conta)
- Depois compartilhar o método de pagamento com as contas de anúncio
- Demonstra ao Facebook gestão profissional

### ETAPA 11: Criar Linha de Negócios e Grupo de Ativos

- Linha de negócios: agrupa contas e ativos
- Grupo de ativos: organiza por marca/região/equipe
- Atribuir **todos** os ativos (página, conta, pixel, Instagram, app) ao grupo

### ETAPA 12: Adicionar Parceiro (Outro BM)

- Se tem outro gerenciador, adicionar como parceiro
- Compartilhar ativos entre BMs dá relevância

## O Que Acontece Após Verificação

- Status: "Não Verificado" → **"Verificado"**
- Limite de contas de anúncio **aumenta** (de 1 para mais)
- Possibilidade de certificações do Facebook
- Acesso a treinamentos específicos
- Possibilidade de créditos
- Gerente de contas olha com mais seriedade
